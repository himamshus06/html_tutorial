CSS Text, Fonts, Margins, Padding, and Borders



CSS Text



CSS text properties allow you to style and control the appearance of text.



Common Properties:

  • color: Sets the color of the text.
  • text-align: Aligns the text (left, right, center, justify).
  • text-decoration: Adds decoration (underline, overline, line-through, none).
  • line-height: Sets the height of lines.
  • letter-spacing: Adjusts space between letters.
  • text-transform: Controls text capitalization (uppercase, lowercase, capitalize).


CSS Fonts



CSS font properties define the font type, size, weight, style, and more.

Common Properties:

  • font-family: Specifies the font type (e.g., Arial, "Times New Roman").
  • font-size: Sets the font size (e.g., px, em, %, rem).
  • font-weight: Specifies font thickness (e.g., normal, bold, 100-900).
  • font-style: Applies styles like italic or normal.
  • font-variant: Enables small-caps text.


CSS Margin



The margin is the space outside the border of an element.

Syntax:

  • margin: value; (applies the same margin on all sides)
  • margin: top right bottom left; (applies specific values for each side)


CSS Padding



The padding is the space between the content and the border of an element.

Syntax:

  • padding: value; (applies the same padding on all sides)
  • padding: top right bottom left; (applies specific values for each side)


CSS Border



The border is the line around an element's padding (or content if padding isn't defined).

Syntax:

  • border: width style color;
    • width: Thickness of the border (e.g., 1px, 2px).
    • style: Style of the border (solid, dashed, dotted, etc.).
    • color: Border color.

CSS Properties Explanation

1. CSS Text

CSS text properties allow you to style and control the appearance of text.


        color: red;
        text-align: center;
        text-decoration: underline;
        line-height: 1.5;
        letter-spacing: 2px;
        text-transform: uppercase;

2. CSS Fonts

CSS font properties define the font type, size, weight, style, and more.


        font-family: "Arial", sans-serif;
        font-size: 16px;
        font-weight: bold;
        font-style: italic;
        font-variant: small-caps;

3. CSS Margin

The margin is the space outside the border of an element.


        margin: 20px;
        margin: 10px 15px 20px 25px;

4. CSS Padding

The padding is the space between the content and the border of an element.


        padding: 10px;
        padding: 5px 10px 15px 20px;

5. CSS Border

The border is the line around an element's padding.


        border: 2px solid black;
        border-width: 2px;
        border-style: dashed;
        border-color: red;
